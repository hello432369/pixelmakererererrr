# pixelmakererererrr
pixel maker

这两天gpt-iamge2生图功能话题火热

同时也有一些问题,比如扣图问题,像素画重绘问题

我这边用ai做了扣图工具,像素绘画工具,纯html代码,双击使用用

生图的时候要加上一句话(白色背景,游戏素材细的黑色外轮廓)

举例:
游戏素材,中国元素,室内,2.5d等距,tilemap瓦片,家具组件,
白色背景,游戏素材细的黑色外轮廓
